隠れマルコフモデル¶

概要¶

  • 観測されない（隠れた）状態をもつマルコフ過程である。

モデル¶

\(p(\mathbf{x},\mathbf{z}) = p(z_{1}) \prod_{n=2}^{N}{p(z_{n}|z_{n-1})} \prod_{n=1}^{N}{p(x_{n}|z_{n})}\)

  • 遷移確率 \(p(z_{n+1}|z_{n})\)
  • 観測確率 \(p(x_{n}|z_{n})\)

HMMの問題¶

  • 評価
    • パラメータを既知とした時、観測結果 \(\mathbf{x}\) が得られる確率を求める 評価 問題
  • 復号
    • パラメータを既知とした時、観測結果 \(\mathbf{x}\) を導く可能性の最も高い状態系列を求める 復号 問題
  • 推定
    • パラメータを未知とした時、観測結果から未知パラメータを求める 推定 問題

評価問題¶

\(P(\mathbf{x}) = \sum_{\mathbf{z}} {P(\mathbf{x},\mathbf{z})}\)

を求める。

  1. 網羅的な方法

愚直に全パターンの計算を行う 状態数が3種の場合,以下の計算を27回全ての状態に対して行う

\[P(\mathbf{x} = v_{1} v_{2} v_{1},\mathbf{z} = w_{2} w_{3} w_{1}) = p(x=v_{1}|z=w_{2}) P(z=w_{2}) * p(x=v_{2}|z=w_{3})P(z=w_{3}|z=w_{2}) * p(x=v_{1}|z=w_{1}) P(z=w_{1}|z=w_{3})\]
  1. フィードフォワード・バックフィワードにより求める

復号問題¶

  • 逐次推定
\[p(z_{n}|X_{n}) = \frac{p(X_{n},z_{n})}{p(X_{n})} = \frac{p(x_{n},X_{n-1},z_{n})}{\int{p(x_{n},X_{n-1},z_{n})}dz_{n}} = \frac{p(x_{n}|X_{n-1},z_{n})p(z_{n}|X_{n-1})p(X_{n-1})}{\int{p(x_{n}|X_{n-1},z_{n})p(z_{n}|X_{n-1})p(X_{n-1})}dz_{n}} = \frac{p(x_{n}|z_{n})p(z_{n}|X_{n-1})}{\int{p(x_{n}|z_{n})p(z_{n}|X_{n-1})}dz_{n}}\]
  • 予測
\[p(z_{n+1}|X_{n}) = \int{p(z_{n+1},z_{n}|X_{n})}dz_{n} = \int{p(z_{n+1}|z_{n},X_{n})p(z_{n}|X_{n})}dz_{n} = \frac{\int{p(z_{n+1}|z_{n})p(X_{n},z_{n})}dz_{n}}{p(X_{n})}\]\[= \frac{\int{p(z_{n+1}|z_{n})p(x_{n},X_{n-1},z_{n})}dz_{n}}{\int{p(x_{n},X_{n-1},z_{n})}dz_{n}} = \frac{\int{p(z_{n+1}|z_{n})p(x_{n}|z_{n},X_{n-1})p(z_{n}|X_{n-1})p(X_{n-1})}dz_{n}}{\int{p(x_{n}|z_{n},X_{n-1})p(z_{n}|X_{n-1})p(X_{n-1})}dz_{n}} = \frac{\int{p(z_{n+1}|z_{n})p(x_{n}|z_{n})p(z_{n}|X_{n-1})}dz_{n}}{\int{p(x_{n}|z_{n})p(z_{n}|X_{n-1})}dz_{n}}\]

網羅的な方法によってp(X)を求めるサンプルコード

# coding:utf-8
import numpy as np


class HMM(object):
  def __init__(self):
      self.ro = np.array([1 / 3, 1 / 3, 1 / 3])

      # p(z_{i} | z_{j}) = A[i,j]
      self.A = np.array([[0.1, 0.2, 0.7],
                         [0.7, 0.1, 0.2],
                         [0.2, 0.7, 0.1]])

      # p(x_{i} | z_{j}) = B[i,j]
      self.B = np.array([[0.9, 0.6, 0.1],
                         [0.1, 0.4, 0.9]])

  def start_probability(self, z0):
      return self.ro[z0]

  def transition_probability(self, z, previous_z):
      return self.A[z, previous_z]

  def observation_probability(self, x, z):
      return self.B[x, z]


"""
z= w1 w2 w0
x= v0 v1 v0
"""
z0, z1, z2 = 1, 2, 0
x0, x1, x2 = 0, 1, 0

hmm = HMM()
l = []

for z0 in range(3):
  for z1 in range(3):
      for z2 in range(3):
          l.append(hmm.observation_probability(x2, z2) * hmm.transition_probability(z2, z1)
                   * hmm.observation_probability(x1, z1) * hmm.transition_probability(z1, z0)
                   * hmm.observation_probability(x0, z0) * hmm.start_probability(z0))

print(sum(l))
